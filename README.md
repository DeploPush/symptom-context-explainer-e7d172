Symptom Context Explainer
Understand symptoms before your appointment

─────────────────────────────────────

📁 Step 1 — Go to your app folder

Mac / Linux:
  cd ~/Downloads/symptom_context_explainer

Windows:
  cd %USERPROFILE%\Downloads\symptom_context_explainer

─────────────────────────────────────

⚡ Step 2 — Run this one command

Mac / Linux:
  python3 -m venv venv && source venv/bin/activate && pip install -r requirements.txt && uvicorn main:app --reload

Windows:
  python -m venv venv && .\venv\Scripts\activate && pip install -r requirements.txt && uvicorn main:app --reload

─────────────────────────────────────

🌐 Open → http://localhost:8000
